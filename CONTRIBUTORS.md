## 🙌 How to Contribute

We welcome contributions! Here’s how you can help:

1. Fork the repository
2. Create a new branch (`git checkout -b feature-name`)
3. Make your changes
4. Commit and push (`git commit -m 'Add new feature' && git push`)
5. Open a Pull Request
